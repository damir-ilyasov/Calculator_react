### ИМТ рассчитывается по стандартной формуле: <br> 
ИМТ = вес (кг) / (рост (м) × рост (м))
## Категории ИМТ:
          < 18.5 - Недостаточный вес
          18.5 - 24.9 - Нормальный вес
          25 - 29.9 - Избыточный вес
          ≥ 30 - Ожирение
